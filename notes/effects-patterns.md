# NgRx Effects Patterns

## 1. switchMap
Use when:
- Need to cancel previous API

Example:
Search API

## 2. mergeMap
Use when:
- Multiple API calls allowed

Example:
Load multiple items

## 3. exhaustMap
Use when:
- Ignore duplicate triggers

Example:
Submit button

## 4. concatMap
Use when:
- Maintain order

Example:
Sequential API calls
